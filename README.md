# 🎓 Free AI & CS Certificates 2026

A curated, sortable comparison dashboard of the **15 best free AI and Computer Science certificates** you can earn in 2026 — with durations, issuers, levels, topics, and direct LinkedIn badge links.

🔗 **Live demo:** https://www.perplexity.ai/computer/a/free-ai-cs-certificates-2026-1kYLdDf5S0irZcufoSF4jg

---

## ✨ Features

- **Sort** by certificate name, issuer, or duration (ascending/descending)
- **Filter** by level — Beginner or Intermediate
- **Filter** by topic — AI Fundamentals, Machine Learning, CS, Security, Generative AI, Prompt Engineering
- **Search** by keyword across names, issuers, and descriptions
- **LinkedIn** button on every row to share your badge directly
- **Enroll** button linking straight to each course
- **Dark / Light mode** toggle
- Fully **mobile responsive**
- Zero dependencies — pure HTML, CSS, and vanilla JS

## 🚀 Run Locally

No build step required. Just open `index.html` in any browser, or serve it:

```bash
# Using Python
python3 -m http.server 8000

# Using Node (npx)
npx serve .
```

Then visit `http://localhost:8000`.
